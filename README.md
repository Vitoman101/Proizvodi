# Proizvodi

Za ubacivanje podataka preko baze potrebno je da odete na putanju /index.aspx
Za ubacivanje podataka u Json fajl potrebno je da koristitie putanju /Proizvods/Create
Jednom kad unesete podatke u formu, otvoriti ce se druga stranica sa crvenim dugmetom koje nakon sto pritisnete podaci ce se upisati u output.json fajl
